Metadata

Component metadata

Metadata describes the objects we build. A great example is a component, which has numerous metadata information, like the templateUrl, styleUrls, directives, providers, etc. 

@Component({
    selector: "directives",
    styleUrls: ["./metadata.component"],
    directives: [PokemonDirective],
    providers: [PokemonService],
    templateUrl: "./metadata.component.html"
})
export class MetadataComponent {
    // Code here...
}
